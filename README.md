# Neighborhood Map

Created for Udacity Front-End Nano-Degree program

## To use
1. Download the files into the same folder
2. Using Terminal, navigate to the folder
3. Run `npm install`
4. Run `npm start`
5. Project should open in browser

## Dependancies
1. GoogleMaps API
2. FourSquare API
